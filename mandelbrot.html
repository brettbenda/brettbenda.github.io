Mandelbrot and Julia Set Viewers

Below are two sketches originally written is the Processing programming language. The upper displays the Mandelbrot set, while the lower a Julia set. Each is only running 20 iterations, to be less CPU intensive.

You can use your mouse to navigate, the 'w' key to zoom in, and the 's' key to zoom out.
