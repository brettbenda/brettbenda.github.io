Voronoi Diagram Generator

Link to GitHub Repository

Below are two processing sketches that generate Voronoi diagrams. Every pixel in the canvas is colored according to which point it is closest to. The two sketches use different rules to determine distance. In both, you can click and drag points around as well as add new points.They were not designed to be deployed on the web, so performance may not be optimal in comparison to running them off your computer. If you would like to see it perform as designed, pull from the GitHub repository linked above and run in PDE.



Euclidean Distace: distance calculated by only taking a straight-line path

Manhattan Distace: distance calculated by only taking vertical or horizontal paths
