Voronoi Diagram Generator

Link to GitHub Repository

Below are two processing sketches that generate Voronoi diagrams. Every pixel in the canvas is colored according to which point it is closest to. The two sketches use different rules to determine distance. In both, you can click and drag points around as well as add new points.They were not designed to be deployed on the web, so performance may not be optimal in comparison to running them off your computer. You can use the links below to download them as desktop applications.
