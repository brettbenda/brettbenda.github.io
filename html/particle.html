Particle Simulator

Link to GitHub Repository

Particle Simulator is a small project I worked on over Summer 2017. It provides an environment to see the effects of various forces on the flow of particles. The forces included are:

  • Gravity: a constant force directed in a constant direction
  • Radial: a constant force directed to a single point
  • Newton: a variable force (dependent on distance) directed to a single point

The particle system can be customized with different particle counts and default velocities. Two different particle types are supported. Circle is best when you care about the position of the particle. Line can be used to see both the position and velocity (longer line means higher velocity).
